GIT

O Git é um sistema de controle de versão distribuído gratuito e de código aberto projetado para lidar com tudo, desde projetos pequenos a muito grandes com velocidade e eficiência.

  • sistema de controle de versões
  • gratuito
  • Código aberto
  • rápido
  • sistema distribuído, ou seja, cada usuário pode ter uma cópia do repositório localmente!
  • é o mais usado

comando para ver se temos instalado e qual a versão: git --version

para iniciar um depósito local: git init

para consultar o status dos arquivos: git status

para adicionar arquivo ao staging area / area de preparação:git add intro_git.html

git add ficheiro1 ficheiro2

git add .

quando for para salvar uma versão do programa / fazer commit / controlado snapshot: git commit -m "pequena descrição"


para ver o histórico de commits:git log

git log --oneline --graph

ramificação do código

listar os ramos/filial e indicar em qual estamos git branch

criar uma nova ramificação / branch git branch nomebranch

para trocater de ramo: git switch nomebranch

para controlar o mergegit merge nomebranch
